Detailed Description

ROS 2 node that converts ego-state and trajectory input into Ackermann control commands.

The node runs cascaded lateral and longitudinal PID controllers, applies configurable feed-forward terms, and constrains the resulting curvature and acceleration commands before publishing them as Ackermann drive messages.
